# Fido

Fido is a super AI Commodore PET, written in KickAssembler for the Commodore PET. (Programming by Deadline)

## Instructions:

1-0, q-p: Change Eyes (not implemented yet)

a-l: Change Mouth

z: SWAP Between DOG and CAT mode (not implemented yet)
